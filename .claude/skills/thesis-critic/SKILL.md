# thesis-critic

## Purpose

Attack the current long thesis and surface the strongest failure case.

## Inputs

- current thesis\n- raw evidence\n- prior failed or weak theses if available

## Outputs

- counter-thesis\n- hidden assumptions\n- unresolved concerns

## Rules

- Fight confirmation bias\n- Prefer strongest realistic objections over weak generic cautions\n- Be concrete

## Failure conditions

- Missing critical source data
- Stale timestamps beyond allowed threshold
- Contradictory evidence without explicit uncertainty handling

## Deliverable format

Return:
- status
- summary
- structured fields
- confidence / uncertainty note where relevant
